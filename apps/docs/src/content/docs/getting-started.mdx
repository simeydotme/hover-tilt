## Installation

Install the package using your preferred package manager:

<Tabs>
  <TabItem label="npm">```bash npm install hover-tilt ```</TabItem>
  <TabItem label="pnpm">```bash pnpm add hover-tilt ```</TabItem>
  <TabItem label="yarn">```bash yarn add hover-tilt ```</TabItem>
  <TabItem label="bun">```bash bun add hover-tilt ```</TabItem>
</Tabs>

## Usage

Because `HoverTilt` is available both as a **Svelte Component (`<HoverTilt />`)** and
a **Web Component (`<hover-tilt />`)**, there are two ways to use it;

<br />
<br />

<Tabs syncKey='api'>
	<TabItem label="Svelte">

    	#### Import and Use

    	Import the `HoverTilt` component in your Svelte file:

    	<Code lang="svelte" title="App.svelte" code={`

<script>import {HoverTilt} from 'hover-tilt';</script>

<HoverTilt>Your content here</HoverTilt>
`} />

    	#### Basic Example

    	<Code lang="svelte" title="MyComponent.svelte" code={`

<script>import {HoverTilt} from 'hover-tilt';</script>

<HoverTilt tiltFactor={1.5} scaleFactor={1.1}>
  <div class="card">
    <h2>Card Title</h2>
    <p>Hover over this card to see the effect!</p>
  </div>
</HoverTilt>

  <style>
    .card {
      width: 300px;
      height: 200px;
      background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
      border-radius: 12px;
      padding: 2rem;
      color: white;
    }
  </style>

`} />

</TabItem>
<TabItem label="Web Component">

    #### Include the Script

    	Add the web component script to your HTML page:

    	```html
    	<!DOCTYPE html>
    	<html>
    	<head>
    	  <title>Hover Tilt Example</title>
    	</head>
    	<body>
    	  <script type="module" src="node_modules/hover-tilt/dist/hover-tilt.js"></script>

    	  <hover-tilt tilt-factor="1.5" scale-factor="1.1">
    	    <div class="card">
    	      <h2>Card Title</h2>
    	      <p>Hover over this card to see the effect!</p>
    	    </div>
    	  </hover-tilt>
    	</body>
    	</html>
    	```

    	#### Using a CDN

    	You can also use a CDN to load the web component:

    	```html
    	<script type="module" src="https://cdn.jsdelivr.net/npm/hover-tilt/dist/hover-tilt.js"></script>
    	```

    	#### Basic Example

    	```html
    	<hover-tilt tilt-factor="1.5" scale-factor="1.1">
    	  <div class="card">
    	    <h2>Card Title</h2>
    	    <p>Hover over this card to see the effect!</p>
    	  </div>
    	</hover-tilt>

    	<style>
    	  .card {
    	    width: 300px;
    	    height: 200px;
    	    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    	    border-radius: 12px;
    	    padding: 2rem;
    	    color: white;
    	  }
    	</style>
    	```
    </TabItem>

</Tabs>
