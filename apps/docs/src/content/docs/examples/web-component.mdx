This page demonstrates the **web component** version of Hover Tilt running **completely standalone** - no Framework, no build step, just pure HTML and JavaScript loaded from a CDN.

The web component is loaded via:

<Code
  lang="html"
  title="index.html"
  code={`
<script type="module" src="https://cdn.jsdelivr.net/npm/hover-tilt/dist/hover-tilt.js"></script>
`}
/>

and then applied to the DOM like this:

<Code
  lang="html"
  title="index.html"
  code={`
  <hover-tilt shadow scale-factor={1.2}>
    <div style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);">
      <h3>Basic Example</h3>
      <p>Hover over this card to see the default tilt effect</p>
    </div>
  </hover-tilt>
`}
/>

<div class="demo-container not-content">
  <hover-tilt shadow scale-factor={1.2}>
    <div class="demo-card" style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);">
      <h3>Basic Example</h3>
      <p>Hover over this card to see the default tilt effect</p>
    </div>
  </hover-tilt>
</div>
